# Product Overview

IP-Management is a system designed to manage device IP addresses within IT/OT (Information Technology/Operational Technology) environments at production sites. The primary focus is on controlling network segmentation efforts through VLAN segments.

## Core Purpose
- Manage device IP allocations in industrial/production environments
- Control network segmentation through VLAN management
- Bridge IT and OT network infrastructure management

## Target Environment
- Production sites with mixed IT/OT infrastructure
- Industrial environments requiring network segmentation
- Organizations managing complex VLAN configurations
